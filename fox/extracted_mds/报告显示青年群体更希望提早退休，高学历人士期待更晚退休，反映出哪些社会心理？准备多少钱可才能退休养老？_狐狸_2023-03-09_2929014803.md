这个问题其实涉及到一个特别有趣的心理现象：

一个人的社会地位越高，他的工作幸福感往往越高，而工作幸福感与他愿意工作的时长是成正比的[1]，

也就是说，一个人的工作时长，很大程度上取决于他的社会地位。

所以题主的提问虽然是“高学历人士期待晚退休，青年人期待早退休”，

但这个问题的本质其实是：

已经享受到高社会地位带来的工作幸福感的人期待晚退休，而还在一线战场当牛马的青年们期待早日退休。

不过，是什么心理机制导致了社会阶层对退休意愿的影响呢？

（对论证过程不感兴趣的朋友，可以直接跳到文章末尾看结论哦）

我们不妨来对比一下高、低社会阶层之间，在哪些方面存在区别，

比较直观的体现在于：1.经济收入，2.人际交往，3.积极心态中介，4.闲暇活动

以上4点，显然都与工作幸福感挂钩，那么，哪一项是占据最主导的地位呢？

武汉大学社会保障研究中心在2017年4月开展了一项调研，详细探索了这个心理机制[2]

研究者使用了来自北京大学“985”项目资助、北京大学中国社会科学调查中心执行的中国家庭 追踪调查(ChinaFamilyPaneStudies,CFPS)的数据, 样本覆盖 25 个省、市、自治区2 , 代表了中国 95%的人口。2010 年基线调查共采访 14960 户家庭, 42590 位个人, 全部采用入户面访形式进行, 家户层面累积应答率为 81.25%, 个人层面应答率为84.14%[3]

在调查中，研究者通过问卷的形式对研究对象的工作幸福感进行了验证，并详细采集了经济收入等中介变量

最后得出的数据和计算过程如下：（不感兴趣的朋友可以跳过）

> 对全样本进行独立样本 _t_ 检验表明, 工作的低龄退休老年人幸福感(_M_ 工作 = 3.74, _SD_ 工作 = 1.06)与未工作者(_M_ 未工作 = 3.99, _SD_ 未工作 = 0.99)差异显著, _t_(1058.88) = −5.50, _p_ < 0.001, cohen _d_ = −0.24。子样本中, 工作的老年人幸福感(_M_ 工作 = 3.73, _SD_ 工作 = 1.08)也显著高于未工作者(_M_ 未工作 = 4.01, _SD_ 未工作 = 0.98), _t_(707.66) = −5.33, _p_ < 0.001, cohen _d_ = −0.27。对幸福感进行回归分析的结果显示, 无论全样本(_β_ = −0.21, _t_ = −4.89, _p_ < 0.001)还是子样本(_β_ = −0.23, _t_ = −4.57, _p_ < 0.001), 工作均显著地负向预测城镇低龄退休老年人幸福感(见表 2)。上述结果均与假设 1 相悖。进一步分析发现, 社会地位显著地调节了老年人工作与幸福感的关系。以工作和社会地位为自变量, 幸福感为因变量进行方差分析, 工作主效应显著, _F_(1,3360) = 13.85, _p_ < 0.001; 社会地位主效应显著, _F_(4,3360) = 54.42, _p_ < 0.001; 工作与社会地位的交互效应显著, _F_(4,3360) = 5.14, _p_ < 0.001。交互效应图和简单效应分析表明, 对较低社会地位(社会地位自评为 1、2、3)的老年人来说, 工作者的幸福感显著低于未工作者(_F_ 1(1,3360) = 26.62, _p_ 1 < 0.001; _F_ 2(1,3360) = 20.20, _p_ 2 < 0.001; _F_ 3(1,3360) = 10.07, _p_ 3 = 0.002), 均值差值分别为 0.59、0.44、0.18, 逐渐减小; 对较高社会地位(社会地位自评值为 4、5)的老年人来说, 工作者和未工作者幸福感差异不描述性统计中独立样本 _t_ 检验显示, 工作的城镇低龄退休老年人的年收入较未工作的老年人高 3103 元, _t_(3404) = 4.87, _p_ < 0.001, cohen _d_ = 0.21。为此付出的代价是, 工作的老年人每天拥有的闲暇时间比未工作的老人少 2.05 小时, _t_(1577.01) = −21.22, _p_ < 0.001, cohen _d_ = −0.73; 他们所参与的闲暇活动的种类平均比未工作的老年人少 0.45 种, _t_(1289.03) = −8.55, _p_ < 0.001, cohen _d_ = −0.33。而有工作和未工作的低龄退休老年人在人际交往(_t_(3393) = −0.87, _p_ = 0.39)和积极心态(_t_(3355) = 0.06, _p_ = 0.95)上并无显著差异。中介变量与因变量的相关分析表明：人际交往、积极心态、闲暇时间、闲暇活动种类均与幸福感呈正相关, 经济收入与幸福感关系不显著。据此, 初步判断闲暇在工作与幸福感之间具有显著的中介效应, 其他中介变量的效应尚不明确。

其实我们可以发现一个有趣的点：

工作与幸福感之间，似乎存在着一个负相关关系，

但这种负相关可以被经济收入、人际交往等带来幸福感的事抵消[4]，

但灰色幽默的是——经济收入、人际交往、积极心态，这些看起来似乎对一个人是否能坚持工作至关重要，但根据实际观察来看，这些影响因子加起来，也不过只是“闲暇时间”影响的一半。

这项横跨半个中国的社会心理学研究似乎证明了一点：

当代年轻人们不爱工作，想早点退休，除了经济收入实在太低、人际交往实在太累之外，很大一部分原因还在于：年轻人们实在是太累了，他们真的需要休息。

总结：（精简大白话版）

1.高学历人士为何比年轻人更期待晚退休？

答：这个问题看似是“高学历人士期待晚退休，青年人期待早退休”，

但这个问题的本质，其实和学历关系不大，这个问题的本质是高社会阶层与低社会阶层之间产生的认知差异：

已经享受到高社会地位带来的工作幸福感的人期待晚退休，而还在一线战场当牛马的青年们期待早日退休。

但高学历者不一定是高社会阶层，但高社会阶层的人普遍拥有高学历，

所以直观表现为“高学历人士”期待晚退休。

2.为什么年轻人更期待退休？

答：因为现在的年轻人实在是太累太累了，他们需要休息。

真的不是他们不知人间疾苦，不知道没钱就要挨饿，不知道没钱就扛不起家庭的重担，他们嘴上嚷嚷着“想退休”，其实并不是他们真的准备退休了，只是单纯因为年轻人们实在是太累了。

但无论再累，生活还是要继续啊，所以喊完累，大家还是得继续硬着头皮面对生活。

既然社会现状不允许年轻人们提前撂挑子不干，那在年轻人们偶尔发发牢骚，吐吐苦水时，我们不妨对他们更宽容一点吧。
