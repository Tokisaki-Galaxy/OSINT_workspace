如果你想吸引一个人帮你做某件事，比如让你妈帮你洗碗（滑稽），

其实是有两种研究机制很成熟的方法供你参考的：

1.在做一件很简单的事时，故意表现得很笨拙，然后将这件事做失败；

2.做一件很复杂的事时，故意表现地极其认真，然后成功。

如果你的表演过程被另一个人看完，那他心中很可能会产生参与的想法。

这个现象的机制，目前被学术界简称为：下行社会比较和竞争动机的链式中介作用。

> 社会比较可分为上行社会比较和下行社会比较(即与境况较好或较差的对象比较)

看起来很复杂？其实完全不复杂，你或许曾在视频广告里看到过这样的沙雕广告：

它们的共同特点都是：明明是一个看似很简单的游戏，但广告里的玩家却总是做出让人高血压的操作，然后游戏失败，气的你想顺着网线过去教他玩。

但这类广告背后的故事可能是你不知道的：这类广告有一部分确实是垃圾页游的推广广告，但还有相当一部分，其实是对外经济贸易大学国际商学院在网络发布的社会调查。

调查中，广告里的游戏分为两种结果：玩家做出了正确的操作，游戏成功；玩家做出了错误操作，游戏失败。

除此之外，还引入了游戏难度等变量。

![](./assets/v2-e6bad56cd3da4b04ac6f6f73c62cc173_720w.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='931' height='465'></svg>)![](./assets/v2-9f1651f7d3ada9b822d83205c5702bcc_720w.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='836' height='455'></svg>)

最后根据不同的难度分组，统计不同游戏结果的广告中各自带来的下载数，将结果统计成下载意愿（详细统计和计算方法如下，不感兴趣可以跳过）

> 以可信度、理解程度和性别为协变量, 以下载意愿为因变量进行 2 (游戏结果：成功/失败) × 2 (游   
> 戏难度：简单/困难)的双因素方差分析。结果表明, 游戏结果的主效应显著, _F_ (1, 242) = 11.91, _p =_  
>  0.001, ηp2 _=_ 0.05, 游戏难度的主效应不显著, _F_ (1, 242) = 0.10, _p =_ 0.752。最为重要的是, 游戏结果和游戏难度之间的交互作用显著, _F_ (1, 242) = 4.58, _p =_ 0.033, ηp2 = 0.02。如图 6 所示, 当游戏关卡难度为简单时, 失败组的下载意愿(_M_ = 6.98, _SD_ = 1.74)显著高于成功组(_M_ = 5.19, _SD_ = 2.53), _p_ < 0.001, ηp2 _=_ 0.15; 当游戏关卡难度为困难时, 失败组的下载意愿(_M_ = 6.21, _SD_ = 2.39)与成功组(_M_ = 6.11, _SD_ = 2.35)并无显著差异, _p =_ 0.808。与此同时, 当游戏结果为成功时, 困难组的下载意愿(_M_ = 6.11, _SD_ = 2.35)显著高于简单组(_M_ = 5.19, _SD_ = 2.53), _p_ = 0.037, ηp2 _=_ 0.04; 当游戏结果为失败时, 简单组的下载意愿(_M_ = 6.98, _SD_ = 1.74)显著高于困难组(_M_ = 6.21, _SD_ = 2.39), _p_ = 0.045, ηp2 _=_ 0.03。

单看数据可能还是有点抽象，我们可以尝试将结果绘制成一张坐标图：

![](./assets/v2-0ac2e8633176e5dde85c7e739b65fe52_720w.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='548' height='337'></svg>)实验结果如图所示

我们可以清楚地发现：

在简单组，观察者在看到失败的结果时，下载意愿更强；

在困难组，游戏结果无论是是成功还是失败，下载意愿都不存在显著差异，失败组只比成功组高一点点。

那么，为什么会出现这个现象呢？

**这就涉及到我们人类的本性了** 。

早在上个世纪，就有研究者发现并提出了一个心理机制（Snyder et al., 1976)：个体更倾向于对他人的成功进行外归因(即归因于外界情境因素), 将他人的失败进行内归因(即归因于行动者个人因素）

说得直白一点就是：

当我们看到别人做成功了一件事，我们会下意识地认为是这件事太简单、他的外部条件太优越，而不是认为他本人足够优秀；

当我们自己做成功了一件事，我们通常会认为是自己的能力足够优秀，而不是这件事很简单。

> 事实上，这个结论在本次游戏广告实验里，同样也是成立的。观看完成后, 被试需要对刚刚游戏中玩家的游戏结果进行归因(1 = “由于玩家的个人因素”, 10 = “由于游戏设计本身的因素”)。结果表明, 相较于目睹游戏成功的被试, 目睹游戏失败的被试更倾向于将游戏结果归于内因(即视频中玩家的个人因素导致了游戏结果), 成功组 _M_ = 6.06, _SD_ = 2.88; 失败组 _M_ = 3.00, _SD_ = 2.38; _t_(98) = 5.79, _p_ < 0.001。这一结果与前人研究一致,。

这个实验结果，对我们而言，其实是很具有灰色幽默色彩的。

在我们的教育心理学里，总是在反复强调他人成功对个体的积极影响，我们的社交平台上不知何时开始总是给我们推他人高质量生活的内容。

我们失去了本该有的正常生活氛围，长期沉浸在被某些人刻意营造的焦虑氛围里。

还记得我们文章开头说的吗，社会比较分为上行社会比较和下行社会比较，

我们就长期处于上行社会比较，不知从何时开始。

我们大部分人是意识不到这一点的，我们只是某一天突然发现自己再也不能像少年时那样沾枕头就能入睡了，发现自己总是莫名其妙地焦虑，发现自己不知不觉买了一堆基本用不上的东西。

或许，这样的风气属于大势所趋吧，为了刺激消费，为了经济的繁荣，为了社会的发展。

但我还是祝各位都能成为一个纯粹的人，就像当初年少时的你那样纯粹，在无可奈何的大势所趋下依然保有一分自己的清醒。
