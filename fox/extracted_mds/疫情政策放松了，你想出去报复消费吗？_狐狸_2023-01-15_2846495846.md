告诉大家一个很恐怖但又很无奈的事实：**你没有去报复性消费，很可能是因为你身边死的人还不够多。**

大家可能都以为“开放”是最近几个月的事情，所以相关研究应该大多聚焦在近段时间。但事实是：关于开放对消费的影响的相关研究，早在2020年初就已经陆续开始了。

由于关注领域不同，我并不了解经济学的研究成果，但可以为大家分享一下心理学研究的主流命题之一：**死亡凸显** 对消费者消费选择偏好的影响。

> 死亡凸显(mortality salience)它是指将个体暴露在死亡信息下, 以此来强迫唤起个体的死亡意识、引发其对死亡的思考。

我们会不可避免地在新闻报道或社交媒体上看到各种各样包含死亡线索的资讯, 尤其是在当前疫情背景下, 社交媒体每天发布的大量新冠肺炎疫情相关资讯、实时更新的死亡和感染数据等等, 这些都使得我们或主动或被动地暴露在死亡信息之中。

而**死亡信息** 的频繁威胁，会引发消费者的恐慌和焦虑, 并进一步导致了他们的消费行为发生显著变化(Pyszczynski et al., 2021)

背后的机制其实研究得已经相对透彻了：

死亡凸显导致了控制感缺失、自尊感下降、死亡焦虑增加等, 从而影响了其后继的消费行为。

说具体一点就是：死亡凸显会导致个体本身的意义系统遭到破坏, 为寻求意义恢复, 个体会更多选择体验性消费。

可能有点抽象，我们可以举个略显极端的例子：

很多被确诊了绝症的患者，由于发现得过于晚，连医院都已经不建议进行治疗了。他们中的一部分会选择结束自己的生命，但更多的人会有另一种选择：进行自己曾经难以承担的消费。

不只是经济上难以承担的消费，比如去奢侈品店购物以及环球旅行等，还有着精神上的难以承担（相对于他曾经而言），比如蹦极、跳伞等。

我们可以发现：死亡凸显会让个体会更多选择体验性消费，死亡凸显离个体越近，消费行为会越明显和夸张。

我们上面举的“绝症患者”的这个例子，属于死亡凸显离个体最近的情况，因为死亡直接作用于个体本身；

而疫情期间则是：我们不断从网络中接收关于死亡的信息，我们周围不断有体弱的人因感染而死去，**也许我们没有在自己身上感受死亡，但死亡却环绕着我们** 。

我们的消费当然不会像绝症患者那么疯狂，但依然会出现“报复性消费”的情况。

暨南大学管理学院在《心理学报》上发表了一项研究，详细解释了这个理论。

他们将消费行为分为实物型消费和体验性消费两种：

如果你的消费更倾向于实物型消费，这其实是一种对生活的可持续性发展抱有乐观态度的体现，说明你对未来是有所期望的，你还在关注消费所买物品的性价比、使用寿命，以及这项消费具体会为你带来什么；

而体验性消费比如蹦极，只带来短时间内的感受，并没有实质性的消费实体。如果你更倾向于体验性消费，说明你更在意当前的实际感受，至于是否会对未来造成不良影响，已经不那么重要。

实验中, 研究人员将被试随机分配到不同的组别，进行死亡凸显或者非死亡凸显的启动，分别进行了死亡凸显和非死亡凸显的心理暗示，并使用PANAS 量表确认暗示是否有效。最后分别统计两组的消费意愿。

得到的数据以及计算过程如下：（不感兴趣可以跳过）

> 首先对死亡凸显启动进行操纵检验。独立样本 _t_ 检验数据结果显示, 死亡凸显组 (_M_ = 5.62, _SD_ = 1.32)引发的对死亡的思考显著多 于非死亡凸显组(_M_ = 3.21, _SD_ = 1.87), _t_(250) = 11.79, _p_ <0.001, _d_ = 1.61, 因此死亡凸显操纵成功。其次对消费类型启动进行操纵性检验。对两种相机的消费属性进行配对样本 _t_ 检验, 结果显示相机 B (_M_ =5.07, _SD_ =1.59)的体验性评价得分显著高于相 机 A (_M_ = 2.88, _SD_ =1.75), _t_(251) = −12.67, _p < _0.001, _d_ = 1.31。这说明消费者更倾向于将相机 B 判断为体验性消费, 而将相机 A 为实物性消费。因此, 消费类型操纵成功。控制变量的检验。对被试的正负性情绪得分进行独立样本 _t_ 检验发现：两组在正性情绪(_t_(250) = −1.55, _p_ = 0.123)上不存在显著差异, 但在负性情绪(_t_(250) = 3.42, _p_ = 0.001, _d_ = 0.17)上存在显著差异, 死亡凸显组的被试报告了更高的消极情绪, 这说明死亡凸显引发了被试短暂的即时情绪反应。虽然分心任务已经起到了很好的屏蔽作用(Greenberg et al, 1994; 2003; 柳武妹 等, 2014; 王鹏 等, 2019), 为了进一步排除情绪可能产生的干扰作用, 参 考Schindler 等(2019)的做法, 将消极情绪作为中介变量进一步分析, 发现情绪的中介作用不显著, 95% CI 为[−0.256, 0.046], 包含 0。此外, 消极情绪对死亡凸显与体验性消费偏好之间的调节作用也不成立(95% CI [−0.325, 0.858], 包含 0)。基于此, 可以排除情绪作为影响体验性消费偏好的备择解释。主要假设检验。独立样本 _t_ 检验发现, 死亡凸显组的体验性产品选择偏好(_M_ = 6.76, _SD_ = 3.11)显著高于非死亡凸显组(_M_ = 5.83, _SD_ = 2.49), _t_(250) = 3.13, _p =_ 0.002, _d_ = 0.33。进一步分析发现, 死亡凸显组(_M_ = 3.98, _SD_ = 1.54)和非死亡凸显组(_M_ = 4.19, _SD_ = 1.61)在实物性消费上的购买意愿不存在显著差异, _t_(250) = −1.03, _p =_ 0.31; 而死亡凸显组 _M_ = 5.40, _SD_ = 1.43)对体验性消费的购买意愿显著高于非死亡凸显组(_M_ = 5.01, _SD_ = 1.55), _t_(250) = 2.07, _p =_ 0.04, _d_ = 0.26。这表明死亡凸显会增加消费者对体验性消费的选择偏好。进一步将死亡凸显作为自变量, 体验性消费选择偏好作为因变量进行多元有序 logistics 回归分析, 结果发现, 死亡凸显对体验性消费选择偏好的影响显著, _B_ = 0.64, _SE_ = 0.23, Wald 2 = 7.60, _p_ = 0.006, 这表明死亡凸显会使人们对体验性消费产生更大的选择偏好。基于此, 假设 1 得到验证。补充分析。假设 2 提出死亡凸显后的消费者更倾向于通过体验性消费来补偿其生命意义感。因此, 对体验性消费是否会提高生命意义感进行了事先检验和分析。结果发现, 死亡凸显组(_M_ = 4.04, _SD_ = 1.67)和非死亡凸显组(_M_ = 4.20, _SD_ = 1.59)对实物性消费的生命意义感知无显著差异, _t_(250) = −0.79, _p_ = 0.43; 而死亡凸显组(_M_ = 5.43, _SD_ = 1.37)对体验性消费的生命意义感知显著高于非死亡凸显组(_M_ = 5.02, _SD_ = 1.50), _t_(250) = 2.27, _p_ = 0.024, _d_ = 0.29。这表明消费者在实物性消费和体验性消费的生命意义感知上确实存在差异, 体验性消费行为很有可能是消费者死亡凸显后进行意义补偿的方式之一。

现在我们可以清楚地发现：在进行了死亡凸显的相关心理暗示后，被试对体验性消费的倾向性有了很显著的提升。

说简单一点就是：为了缓冲死亡带来的恐惧及焦虑情绪，为了找回被死亡冲碎的生命意义感，我们选择了消费，这个消费甚至不一定会有实物。

这就是报复性消费，**以我们对死亡的恐惧为基础的消费** 。

当然，被这种“报复性消费”的心理影响是一回事，而是否做出实际行动又是另一回事。很多人因为横遭不幸，确实产生了这样的消费心理，但他作为一位坚强的父母、子女，坚定地履行了自己的家庭义务，克制住了这种消费心理。

但如果你甚至感受不到这种报复性消费对你的影响，那么恭喜你，你可能是这个时代的幸运儿。

你的父母，你的儿女，你的爱人，他们没有在这场天灾中丧生，死亡对你的危机感仅停留在社交媒体和口口相传，还没有来到你的身边。

你的父母，你的儿女，你的爱人，他们是我们通向死亡的最后一道屏障。当你和他们一起边看电视边吃饭时，你看着他们的笑容，听着他们嘴里琐碎的家长里短，你会感觉死亡是一件离我们很遥远的事。

即使永别在第二天就可能会到来，但直到前一天的夜晚，你都睡得如此的安心。

2023年，祝我们都能活下去。
