每次看到有人分享自己奉承领导的经验，并大肆强调讨好领导的重要性时，我都会思考一个问题：

**与其老老实实地等领导来PUA我们，我们为什么不整点骚操作，反向PUA一下我们的领导呢？**

当然，这是一项庞大的学习任务，涉及到一系列的心理学问题，我们本次不妨先从最基础的部分开始：如何根据一个人性格特质，初步预判他的行为倾向和奖惩心理。

不过在开始之前，我们必须明确一点：

我们努力地尝试与领导建立关系，本质上是希望获得来自领导的资源和精神回报，

而与领导建立关系的方式，无非也就是两种——

完全能拿上台面的社交，比如公事公办时的必要交流，正常社交场合的聊天；

不能拿上台面细说的社交，比如你利用职务之便帮领导省了麻烦，或者撒了点小谎，帮领导蒙混过关。

可关键在于：“领导”是一个样本量极其丰富的群体，并非每一个顶着领导头衔的人都是一个模子刻出来的，每一个“领导”都有着自己的特色和倾向，

有的领导就喜欢下属公事公办，人家不吃溜须拍马那一套，而有的领导就对你的奉承很受用。

那么问题来了，我们应该以怎样的方式去面对怎样的领导，才能更好地建立我们与领导间的关系，让我们的努力更容易激发领导的感激之情，以获取资源或精神上的回报呢？

（对研究和证明过程不感兴趣的朋友，可以直接跳到文章末尾看结论哦）

  


  


  


  


在组织中, 下属做出非伦理行为可能出自利他动机。例如, 下属通过撒谎来保护自己的领导, 隐瞒可能损害领导声誉的信息等[1]，。这类行为被称为亲领导非伦理行为(Unethical Pro-supervisor Behavior, UPSB），指下属维护领导利益但违反法律法规或社会道德规范的行为，

目前为止，学界对 UPSB 的后果(尤其是领导反应)其实还是众说纷纭，不过根据目前的研究进展,我们其实不难发现： 领导反应是治理下属 UPSB 的关键因素。

第一, 下属 UPSB 的直接对象是领导[2], 目的是为了使领导受益, 因而, 下属更在意领导对 UPSB 的反应并据此决定后续是否从事该行为，

第二, 领导通常具有一定的资源分配权力(如奖惩), 决定下属在工作中的收获与发展, 从而塑造了下属的态度与行动[3]，

鉴于此, 我们不妨讨论一下领导对下属 UPSB 的反应机制及其如何影响下属后续UPSB 变化，以反向推理下属的UPSB会对领导产生的影响。

不过在讨论时，我想强调一点：同样是领导和下属之间的互动，不同的文化背景的群体会有其独有的特殊性，比如在中国情境下人际交往中就有它独有的特色，

所以在探讨中国情境下领导对下属 UPSB 的反应机制时, 我想引入一个“报”视角，

“报”这个概念最初由刘兆明教授在1992年提出，它既强调互动双方“有来有往”的交互补偿[4], 又涵盖了积极“报”和消极“报”[5]。

因为 UPSB 兼具“亲领导”和“非伦理”双重属性, 所以作为获益者的领导既可能会给予下属正面的资源性“报”, 也可能会对下属实施负面的惩罚性“报”。

究竟何种“报”会占据上风, 这受到道德机制和伦理规范的影响[6] 如领导的正直程度。

高正直的领导格外关注下属 UPSB 的“非伦理”属性, 不会为了一己私利而违反道德准则, 故对下属 UPSB 进行消极评价和回应[7], 实施具有惩罚性质的消极“报”, 由此降低了下属继续做出 UPSB 的可能性。

相反, 低正直的领导会模糊 UPSB 的“非伦理”属性, 关注“亲领导”属性, 故对下属 UPSB 做出积极“报”, 从而增加了下属继续做出 UPSB 的可能性。

鉴于此, 我们似乎能够以领导“报”为切入点, 考察领导对下属 UPSB 的反应以及此种反应对下属后续 UPSB 的影响,进而反向推论下属UPSB对领导的影响。

首都经贸大学劳动经济学院 和 广州大学管理学院 、吉林大学商学与管理学院联合开展了一项研究，以这个视角为切入点，详细讨论了上文提到的假设。

研究者招募 了总计240名具有实习经历的商科生(年龄为 19~21 岁)，随机分配到对应的高正直度的领导和低正直度的领导处，形成领导−下属配对样本领导扮演者与下属扮演者在实验室的两个不同房间(隔音效果较好, 配有电脑与智能手机)内进行办公。被试双方通过智能手机中的微信软件进行交流。

（悄悄说一句，这个实验我感觉相当眼熟，似乎是改编自 Fehr 等(2019)的亲组织非伦理行为情景材料[8]）

在被试熟悉情景信息后, 领导与下属扮演者均将收到 UPSB 的实验操纵材料。在接受 UPSB 操纵材料后, 领导扮演者需要填答心理测量问卷(感激与愧疚)并根据填答感受给下属扮演者发送微信信息。实验者向领导扮演者提供了参考信息句子(包括惩罚和回报)并告知其可在与下属微信沟通过程中使用这些句子信息(但不限于此)。在微信互动沟通环节结束后, 邀请下属填写问卷, 报告感知的领导惩罚和资源回报。最后, 邀请下属扮演者决定下个季度是否仍然采购该原材料(对购买者有潜在风险但有利于领导的考核与晋升)的决定。

研究者通过实验操纵与UPSB 改变量的测量进行检验，并同时进行领导正直的测量，领导资源回报与惩罚的测量。

研究者得出的结果如下图所示：

![](./assets/v2-07905eef290d02186efb5aa86d6aeace_720w.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1076' height='585'></svg>)路径分析结果，注：N = 277, ***p &lt; 0.001, **p &lt; 0.01, *p &lt; 0.05

我们其实可以清楚地发现：

1.领导正直在下属初始 UPSB 与领导感激之间起显著负向调节作用。

> 简单斜率分析显示, 当领导正直水平低时, 下属初始 UPSB 对领导感激的影响更强(_b_ = 0.51, _t_ = 8.16, _p_ <0.001); 当领导正直水平高时, 下属初始 UPSB 对领导感激的影响更弱(_b_ = 0.22, _t_ = 3.79, _p_ < 0.001)。，

2.领导感激正向影响领导资源回报(_b_ = 0.36, _t_ = 5.60, _p_ < 0.001)

3.领导资源回报正向预测下属 UPSB 改变量(_b_ = 0.61, _t_ = 8.44, _p_ < 0.001)

4.与高正直的领导相比(_b_ = 0.04, 95% CI [0.02, 0.08], _p_ = 0.004), 低正直的领导下(_b_ = 0.10, 95% CI [0.06, 0.17], _p_ < 0.001)下属初始 UPSB 经由领导感激与领导资源回报对下属 UPSB 改变量产生更强影响。

5.领导正直在下属初始 UPSB 与领导愧疚之间起显著正向调节作用

> 简单斜率分析显示, 当领导正直水平高时, 下属初始 UPSB 对领导愧疚的影响更强 (_b_ = 0.43, _t_ = 8.07, _p_ < 0.001); 当领导正直水平低时, 下属初始 UPSB 对领导愧疚的影响更弱(_b_ = 0.16, _t_ = 2.11, _p_ = 0.035)

6\. 领导愧疚正向影响领导惩罚(_b_ = 0.36, _t_ = 6.46, _p_ < 0.001)

7.领导惩罚负向预测下属 UPSB 改变量(_b_ = −0.37, _t_ = −5.35, _p_ < 0.001)

8.与低正直的领导相比， 高正直的领导下，下属初始 UPSB 经由领导愧疚与领导惩罚对下属 UPSB 改变量产生更强的抑制作用。

总结：（精简版）

我们通过非伦理行为（UPSB ，比如通过撒谎来保护自己的领导, 隐瞒可能损害领导声誉的信息、用财物贿赂领导）来影响领导的心理过程，进而让他们对我们产生资源回报的行为，本质上是通过以下这个模型来进行的。

![](./assets/v2-a2006fb3e01c67abe1157082d92f5c88_720w.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='530' height='149'></svg>)

当你的行为成功触发了领导的“感激”时，你会更有可能获得来自领导的资源回报，

当你的行为触发了领导的“愧疚”时，更可能会得到来自领导的惩罚。

在过去，很多人喜欢自我感动，认为只要勇于牺牲、勇于奉献，愿意大出血去贿赂，就一定能获得好的回报，

但想象终究只是想象，现实中的真实情况却截然相反：一个人是否愿意对你投桃报李，与你自我牺牲、奉献的程度并没有太大的相关性，关键是你的行为引发了他怎样的情绪，是让他感到感激，还是让他感到愧疚，

比如同样是你去花重金贿赂，有的人受贿后会觉得你这人真不错，能处，

有些人哪怕你再花10倍的价格去贿赂，他也会觉得你在侮辱他，进而对自己的过去行为感到惭愧。

而导致这两种领导产生不同行为的因素，最重要的是领导的“正直度”，

领导的正直度越低，下属UPSB越容易触发领导的感激，你会更有可能获得来自领导的资源回报，

领导的正直度越高，下属UPSB越容易触发领导的愧疚，你会更有可能获得来自领导的惩罚，

所以在你有任何“进步行为”之前，请务必把前期工作做好，免得你还在窃窃自喜的时候，大佬已经开始琢磨着一巴掌拍死你拿你脑袋祭旗。
