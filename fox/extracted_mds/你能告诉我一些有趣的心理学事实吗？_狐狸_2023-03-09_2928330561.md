当你试图和几位对手竞争同一个岗位时，如果你们实力和条件相近，但你依然想从中脱颖而出，成为竞争的胜利者，

那除了提升自身的硬实力，其实还有一个另辟蹊径的方法供你参考：

不再局限于与竞争对手的勾心斗角，而是从**面试官的心理** 入手，从根本上解决问题。

具体的实施方法和理论基础，其实已经有相对成熟的心理学研究，

先说结论：

先提前调查清楚面试官的擅长领域，然后在面试时的自我介绍、能力展示的环节里，着重介绍**与面试官擅长领域重合的领域的能力** 。

（对论证过程不感兴趣的朋友，可以直接跳到文章末尾看结论哦）

面试官擅长什么，我们就着重介绍自己在那些方面的能力，居然会有利于面试？

这似乎是一个很反直觉的结论，

有个成语叫“班门弄斧”，即外行人在内行人的面前卖弄才华，最后贻笑大方，

所以在我国的文化背景下，为了避免失败和尴尬，绝大多数人在展示自己时，会选择尽可能“避开”面试官的擅长领域，以免被在该领域更专业的面试官挑出错误，进而导致面试失败。

这个思路似乎看起来没什么逻辑漏洞，但关键问题在于：这是我们脑补的面试官心理，而非真实的面试官心理，

而不同人的社会认知和行为，是会深受其所处情境和角色影响的，[1]我们以为面试官会讨厌班门弄斧的人，但实际情况到底是怎样的呢？

华东师范大学心理与认知科学学院在2023年1月进行了一场研究[2]，对上述结论进行了验证。

研究者设计了8个实验方案，受限于篇幅，在此仅列举一项。

研究者介入了多场 多学科的征文竞赛（下文以语文、数学为例）。候选人提交的作品将与主动提交另一学科论述的参赛者作品一同呈现给比赛的评委, 评委从中选出一名胜者。

候选人接着阅读两份分别关于语文、数学学习的论述, 得知这两份论述为往年比赛作品, 为了节省时间, 自己可直接从中选择一份, 无需自行写作, 此外第三方学校老师认为这两篇论述比较空泛, 质量一般。

候选人了解到, 评委自认为擅长语文、不太擅长数学(或擅长数学、不太擅长语文), 会在生活中感受到自己擅长语文(或数学)。随后候选人回答“请你预测, 你提交哪一学科的论述, 更有可能获胜？” 

评价者首先回答自己擅长哪门学科(语文或数学), 回想并写下在生活中感到自己语文或数学不错的瞬间。由此, 研究者区分出擅长不同领域的评价者。接着, 评价者需作为征文比赛的大众评委, 从两名提交不同学科学习论述的参赛选手中选出一位胜者。选手A提交了语文学习论述, 选手B提交了数学学习论述, 第三方学校老师认为这两篇论述比较空泛, 质量一般。评价者阅读两篇论述后回答“你会选择谁获胜”

最后得出的实验数据以及计算过程如下：（不感兴趣的朋友可以跳过哦）

> 二元逻辑回归结果显示, 角色(0 = 候选人, 1 = 评价者)能显著预测录取选择(0 = 班门弄斧者, 1 = 不班门弄斧者) (_B_ = –1.50, _SE_ = 0.30, Wald χ2 = 25.43, _p_ < 0.001, _Exp_(_B_) = 0.22, 95% CI =[0.13, 0.40]), 无论小赵被分配到评价者擅长还是不擅长领域, 候选人都低估谈及评价者擅长领域后被录取的可能性(小赵被分配到评价者擅长领域：χ2 (1, _N_ = 130) = 9.52, _p_ = 0.002, φ = 0.27; 小赵被分配到评价者不擅长领域：χ2 (1, _N_ = 130) = 18.38, _p_ < 0.001, φ = 0.38)。分配领域的主效应不显著(_B_ = –0.20, _SE_ = 0.28, Wald χ2 = 0.50, _p_ = 0.479, _Exp_(_B_) = 1.22)。角色和分配领域的交互效应不显著(_B_ = –0.49, _SE_ = 0.59, Wald χ2 = 0.67, _p_ = 0.413, _Exp_(_B_) = 0.62)。角色和赛制的交互作用显著(_B_ = 3.45, _SE_ = 0.55, Wald χ 2 = 39.67, _p_ < 0.001, _Exp_(_B_) = 31.62, 95% CI = [10.80, 92.63])。而高达 70.31%的评价者让选自己擅长领域的选手晋级, χ2 (1, _N_ = 128) = 24.52, _p_ < 0.001, φ = 0.44。淘汰赛中, 75.38%的候选人认为选择评价者擅长领域更可能被淘汰, 而只有 38.81%的评价者淘汰选自己擅长领域的选手, χ2 (1, _N_ = 132) = 17.99, _p_ < 0.001, φ = 0.37。

其实我们可以发现：无论这场比赛的赛制是“晋级制”还是“淘汰制”，评价者都更倾向于选择让自身擅长领域的被试通过比赛，

仅有38.81%的评价者淘汰选自己擅长领域的选手。

不过有趣的是，当赛制为晋级制时，大部分的被试更愿意展示自己擅长的领域，而非评委擅长的领域，

而赛制为淘汰制时，被试们更愿意投其所好，选择评委擅长的领域作为自己的能力展示。

评委会产生这种心态的心理机制其实很有趣，尽管他需要评价被试的能力, 但是据情绪渗透模型(affect infusion model)和情感信息理论(feelings-as-information theory)[3], 人的情绪会影响其评价[4]，

当被试选择班门弄斧时, 能够让评价者想到自己擅长的领域, 进而联想到在该领域所得成就及认可, 感受到自我实现的价值, 唤醒自豪感[5], 这驱使评价者在评价候选人时秉持积极态度, 对其作出正面评价[6]，

从直观结果来看，就是班门弄斧的被试，更容易得到评选人的青睐，进而从竞争者中脱颖而出。

总结：（精简大白话版）

如果你希望从一批和你实力相近的候选人中脱颖而出，成为面试、考试的赢家，可你又来不及提升自己的硬实力，那你能做的就是从评委身上入手，

具体方法以及步骤如下：

1.调查面试官的擅长领域或学术背景，

2.根据面试官的擅长领域，将你面试时能力展示的重点放在该领域。在论述时，你需要表现出专业的地方，但也不用特别刻意地遮掩自己的不足之处。

在行业大佬面前卖弄该行业的才华，看似是“班门弄斧”，会造成大佬的蔑视和不屑，但真实情况往往正好相反：

当你选择班门弄斧时, 能够让评价者想到自己擅长的领域, 进而联想到在该领域所得成就及认可, 感受到自我实现的价值, 唤醒自豪感， 这驱使评价者在评价候选人时秉持积极态度, 对其作出正面评价。
